# Bug Report – Sample

**Title:** Login button does not respond after invalid login attempt

**Environment:** Android – UAT

**Steps to Reproduce:**
1. Open login screen
2. Input invalid credentials
3. Tap login button
4. Try login again with valid credentials

**Actual Result:**
Login button is unresponsive

**Expected Result:**
User should be able to retry login

**Severity:** High  
**Priority:** High
